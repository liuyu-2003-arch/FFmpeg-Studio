# FFmpeg Studio Ultimate

FFmpeg Studio Ultimate 是一个基于 Web 的图形用户界面（GUI），旨在简化 `ffmpeg` 命令行工具的常见操作。它提供了一个用户友好的界面，用于生成复杂的 `ffmpeg` 命令，而无需手动编写脚本。

## ✨ 功能

- **直观的拖放界面**：轻松添加和重新排序要处理的媒体文件。
- **三种操作模式**：
    1.  **合并与章节**：将多个剪辑合并为一个文件，并自动生成章节标记。
    2.  **转换为 MP3**：将视频或音频文件批量转换为 MP3 格式。
    3.  **嵌入字幕**：将 `.srt` 字幕文件嵌入到相应的视频文件中。
- **动态命令生成**：当您添加文件、选择模式和设置选项时，应用程序会实时生成 Shell 脚本。
- **简化的工作流程**：只需设置您的工作目录，添加文件，然后复制生成的命令。
- **无需安装**：作为一个单页 HTML 应用程序，它直接在您的浏览器中运行，无需后端或服务器。

## 🚀 如何使用

1.  **打开 `index.html`**：在您喜欢的 Web 浏览器中打开 `index.html` 文件。
2.  **设置工作目录**：
    - 在“粘贴 Finder 路径...”输入框中，粘贴您媒体文件所在的绝对路径。
    - 该字段将验证路径是否存在。
3.  **添加媒体文件**：
    - 将您的媒体文件直接拖放到应用程序窗口中。
    - 或者，单击拖放区域以打开文件选择器。
4.  **选择操作模式**：
    - 单击窗口底部的选项卡以选择您要执行的操作：
        - **合并+章节**
        - **转 MP3**
        - **嵌字幕**
5.  **自定义输出（如果适用）**：
    - 在“合并+章节”模式下，您可以为合并后的文件指定输出文件名。
    - 您还可以为每个输入文件编辑章节标题。
6.  **生成并复制命令**：
    - 一旦提供了有效路径并添加了文件，右侧面板将展开以显示生成的 Shell 脚本。
    - 单击 **复制命令代码** 按钮。
7.  **执行脚本**：
    - 打开终端。
    - 粘贴复制的命令并按 Enter。
    - 该脚本将在您的工作目录中执行 `ffmpeg` 操作。

## 🛠️ 操作模式详解

### 1. 合并+章节

此模式旨在将多个视频或音频文件合并为一个连续的文件。它还会根据每个剪辑的持续时间自动创建章节。

- **工作原理**：它生成一个脚本，该脚本创建一个 `inputs.txt` 文件用于 `ffmpeg` 的 `concat` demuxer，并创建一个 `metadata.txt` 文件来定义章节。
- **用例**：将多个视频剪辑（如相机录像或下载的剧集）合并为一个带有导航章节的单一文件。

### 2. 转 MP3

此模式可快速将一批媒体文件转换为 MP3 格式。

- **工作原理**：对于每个输入文件，它都会运行一个 `ffmpeg` 命令，提取音轨并将其编码为高质量的 MP3。
- **输出**：每个 MP3 文件都保存在与原始文件相同的目录中，并带有 `.mp3` 扩展名。

### 3. 嵌字幕

此模式将 `.srt` 字幕文件嵌入到具有相同基本名称的视频文件中。

- **工作原理**：对于每个视频文件，它会检查是否存在相应的 `.srt` 文件（例如，`my_video.mp4` 和 `my_video.srt`）。如果找到，它会使用 `ffmpeg` 将视频、音频和字幕流复制到一个新的 Matroska (`.mkv`) 容器中。
- **输出**：新文件以 `_sub.mkv` 后缀保存。
